## Synechron Challenge Book
### Rails Start using Postgresql
```bash
git clone https://github.com/mrrobot16/synechron-book
cd synechron-book
bundle install
rails db:create
rails db:migrate

Create Books
rails db:seed
```

### Angular2 Start
```bash
cd client
npm install
npm start

Once all files bundle
do
cd .. && rails server
Open it in your browser (http://localhost:3000)
```
